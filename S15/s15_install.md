## Sommaire

1. Installation de iRedMail

Tout d'abord :

- `wget https://github.com/iredmail/iRedMail/archive/refs/tags/*.tar.gz`
- `tar -xzf *.tar.gz`
- Se place dans le dossier avec `cd 1.6.8`
- `bash iRedMail.sh`
